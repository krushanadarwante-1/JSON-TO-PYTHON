# Beginner-Friendly Data Analysis Project

## 📌 Project Overview
This is a small but beginner-friendly **Data Analysis project** built entirely with **pure Python (no pandas, no NumPy)**.  
It demonstrates how to clean and analyze data using lists, dictionaries, sets, and the `json` module.

The project includes:
- Handling missing values
- Removing duplicate or inconsistent data
- Converting JSON data to Python structures
- Suggesting pages users might like
- Suggesting friends users may know

---

## 📂 Folder Details
- **HandleMissing-RemoveDuplicate**  
  Scripts for cleaning the dataset:
  - Handle missing values
  - Remove duplicate or inconsistent data
  - Standardize the data format

- **Json-To-Python**  
  Scripts to load JSON data into Python and display user connections.

- **Pages_You_Might_Like**  
  Recommendation system that suggests pages a user might like based on mutual interests.

- **Suggesting_Friends**  
  Friend suggestion system that recommends people a user may know using mutual connections.

---

## 🛠 Features
- Pure Python implementation (no pandas, no NumPy)
- Beginner-friendly data analysis logic
- Uses built-in Python data structures
- Simple recommendation engines
